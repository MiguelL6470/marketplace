CUSTOS DE INFRAESTRUTURA (Mensais)
-----------------------------------

1. HOSPEDAGEM - VERCEL PRO
   - Vercel Pro: R$ 117,00/mês (cotação USD/BRL ~R$ 5,85)
   - Necessário para produção (SSL grátis, CDN, analytics)
   - Plano Free: R$ 0,00 (não recomendado para produção)

2. BANCO DE DADOS - NEON POSTGRESQL
   - Neon Starter (Pay-as-you-go): R$ 30,00/mês
     * Inclui 0.5 GB de storage + 100 horas compute
   - Neon Launch (Fixo): R$ 79,00/mês
     * 10 GB storage + 200 horas compute
   - Neon Scale (Fixo): R$ 199,00/mês
     * 50 GB storage + 500 horas compute
   
   Recomendação para MVP: Neon Starter (R$ 30/mês) - suficiente para começar

3. ARMAZENAMENTO DE IMAGENS - SUPABASE STORAGE
   - Free Tier: R$ 0,00/mês
     * 1 GB storage + 2 GB bandwidth
   - Pro (se necessário): R$ 29,00/mês
     * 100 GB storage + 200 GB bandwidth
   
   Recomendação para MVP: Free Tier (R$ 0) - suficiente para início

4. SISTEMA DE PAGAMENTOS (GATEWAY)

   OPÇÃO A: MERCADO PAGO (Recomendado para Brasil)
   - Taxa por transação (cartão crédito): 4,99% + R$ 0,39
   - Taxa por transação (cartão débito): 3,99% + R$ 0,39
   - PIX: 1,99%
   - Cobrança Recorrente: Mesmas taxas
   - Custo mensal fixo: R$ 0,00 (sem mensalidade)

   OPÇÃO B: STRIPE
   - Taxa por transação: 3,99% + R$ 0,40 (cartão crédito/débito)
   - PIX: Não disponível (apenas cartões)
   - Cobrança Recorrente: Mesmas taxas
   - Custo mensal fixo: R$ 0,00 (sem mensalidade)

   Recomendação: Mercado Pago (melhor para Brasil, suporta PIX)
   
   Observação: As taxas são cobradas apenas sobre transações processadas. Não há custo fixo mensal.
